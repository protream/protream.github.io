练习

下面一些问题留给你做练习:

  1. 从每一行row中按照headers过滤信息，将结果添加到prettytable中

  2. 像开始的图片中那样将出发站和到达站，出发时间和到达时间显示为一列

  3. 添加颜色（提示：使用colorama, termcorlor或ANSI转义字符）

  4. 添加异常处理，如果用户输入的车站或日期有错误怎么办？如果网络异常怎么办？

  5. 添加更多的日期格式支持: 比如用户输入20160701也可以查

  6. 添加参数支持，用户可以指定火车类型
