《如何有效阅读一本书》读书笔记

发表于 2019-02-04 |

“你读过《XXXXX》吗？”
“这本书的主要内容是什么？”
“你最喜欢书中的那个部分？”
“这本书有什么优缺点？”

面对这一连串问题，你是否像我一样，很多时候只能回答第一个问题，比如我读过《时间简史》，但是现在却记不起它讲的任何内容。人有时候过于相信自己的脑袋，认为看过就能记住，但随着时间流逝，一本书我们往往只能记住百之一二，乃至只记住了一个书名。如果明明读过却只记得书名，那和没读过有什么区别？这也是我找到并阅读这本书的原因：

  • 如何让自己读过的书物尽其用，真正耐用，尽量让自己不忘记书的内容，使之融入自己的身心；
  • 如何将书中的信息应用到自己的生活中，在享受读书的过程中塑造自我；

作者提出了他的解决办法——读书笔记法。作者将整个读书的过程分为下图中的几个部分，并对每个部分加以阐述。其中如何读书、写读书笔记是本书的重点，也是对我比较有用的地方，

如何读书

下面是作者读书的大致流程：

  1. 通读全书，将自己认为比较重要、值得记录的页面的右上角折起；
  2. 将折角页面的内容再读一遍，如果还认为这页值得记录，就将右下角也折起；
  3. 再读一遍折起两角的页面，用笔在上面做标记；

如何做标记呢？作者的方式是：

  • 直线：用于“重要”、“客观上很重要”、“暂时先画出”的情况；
  • 波浪线：用于“非常重要”、“主观上觉得重要”、“想记住这里，以后加以运用”的情况；
  • 圆圈：用于圈出重要的专有名词、关键词、关键句等吸引眼球的的内容；

当然，我们也可以创造自己的标记系统，但做标记绝对是值得学习的一件事。

如何做笔记

标记好内容后，下一步就是做笔记了，作者认为：

为了写出精炼而深刻的读书笔记，你应该在读一本书以后，认真回想需要摘抄哪一页、哪一行，思考怎样撰写读书笔记，然后严格筛选需要摘抄的内容，尽量提炼自己的想法，争取用最短的语句表达深刻的含义。

基于这一想法，作者提出了“葱鲔火锅式”读书笔记法，机构如下：

摘抄可以促进对内容的记忆，加深对书的理解。虽然现在的电子时代，作者还是提倡用手写来完成这个过程。

尽管抄完全书也不一定能记住全部内容，但某段文章、某个词、作者的语气和思想等等，都会随着抄写的身体感受被深深地刻印在身体里。

思想输出

相信大家都有过这样的感受，有时候自己认为非常了解的事情或知识，一旦需要向别说起来或写下来，还会出现很多模糊不清的地方。

你在为别人讲解书的内容时，你才能真正理解它。

当我们尝试向对方传达想说的内容时，大脑会自觉地把原来杂乱无章的想法组织起来，这个时候你就会发现缺失的细节，我们才会去调查那些以前未注意到的出处、资料，重新整理自己的思绪、审视自己的观点，建立起系统的知识体系。这大概就是我们要写文章或者博客进行输出的原因了吧，它是对我们知识和思想的一次升级。

如果把读书比作做菜，选书购书是买菜，读书做笔记是处理食材，通过读书笔记进行思想输出就是开火炒菜了。

这一步，作者展示了自己读《浓香与美味的密码》的读书笔记，以及转化成书评，这是一个很好的示例。

小技巧

书中提到了一些一些读书小技巧，我觉得有用的记录如下：

  1. 对于难懂的书，可以从简单的入手，读不懂《经济学原理》，可以先看一遍《漫画经济学》；
  2. 看别人的书评，发现不同的视角；
  3. 养成重读的习惯，经常看看读书笔记，必要时重读原书；
  4. 不间断的坚持。

在做一件的中途如果出现一天空白，或者做事的频率变慢了，就会让人感到失去乐趣。

这一论断不知道是否有科学依据，但从我过往的生活经验来看，确实是这样。

总结

读书是门技术。作者从自己多年的实践中总结除了这些经验和技巧，还是有其可取之之处。总的来说，这些经验技巧比较适用于读知识性、有思想的书，你是希望能从中学到东西或丰富你的思想的，如果只是为了娱乐、消磨时间，倒也不必这么大费周章。
